Introdução

Aula 01 - introdução.js

alert("Fala aê noobada!");
console.log("De boas?");

Há muito tempo que o JavaScript se tornou mais do que uma mera linguagem para fazer truques bobos em páginas HTML e atualmente é praticamente impossível imaginar a internet sem a existência dele. Em conjunto com o HTML e o CSS, o JavaScript - bem como suas bibliotecas, dialetos e tecnologias relacionadas - é um dos alicerces fundamentais da web como a conhecemos.

Embora originalmente o JavaScript tenha sido projetado para rodar em navegadores, atualmente também é possível executar aplicações escritos nessa linguagem em servidores web graças ao Node.js.

Criado em 2009, o Node.js é um conjunto de ferramentas opensource que permite criar servidores web para execução remota de aplicações JavaScript. Serviços importantes como PayPal, LinkedIn e Groupon usam Node.js para funcionar.

Graças aos avanços proporcionados pela comunidade de desenvolvedores dessa ferramenta, existem implementações do Node.js até mesmo para dispositivos da chamada Internet das Coisas: o Tessel, computador semelhante ao Arduino, executa aplicações embarcadas rodando em Node.js.

O JavaScript já transcendeu os navegadores e agora permite fazer virtualmente qualquer software que se possa imaginar, inclusive aplicativos para smartphones!

Todos os sistemas operacionais móveis disponíveis no mercado suportam JavaScript, sendo possível construir aplicativos para Android, iOS, Windows Phone. A vantagem aqui é que geralmente é mais fácil desenvolver aplicativos compatíveis com todas as aplicações, ao contrário do desenvolvimento com linguagens nativas, que limita as opções ao SO de origem (Java para Android, Swift para iOS, etc).

Sencha Touch, PhoneGap, Titanium e outras tecnologias são apenas alguns exemplos de ferramentas que permitem a criação de poderosos aplicativos mobile com mais facilidade e flexibilidade.

Number

Aula 02 - numbers.js

Em JavaScript, há apenas um tipo de número e ele pode ser escrito com ou sem decimais.

var x = 34.00; //Um número com decimais
var y = 34; //Um número sem decimais

Números muito grandes ou pequenos podem ser escritos com notações científicas(expoentes).

var x = 123e5; //12300000 var y = 123-5; //0.00123

O JavaScript usa um número fixo de bits, 64 deles, ou seja, há um quantidade limite de armazenamento que chega a 18 quintilhões. Os números são sempre armazenados como números de ponto flutuante de precisão dupla, seguindo o padrão internacional IEEE 754. Ao contrário de outras linguagens, não há a definição de tipos de números, como por exemplo, inteiros, ponto-flutuante, entre outros, ou seja, não há a necessidade de declarar o tipo de número que você está usando (Como por exemplo em java há double, int, etc).

String

Aula 03 - strings.js

Strings são usadas para determinar textos, e podem ser declaras com aspas simples ou dupla, mas lembre-se: as aspas no início e fim devem combinar.

"Patch my boat with chewing gum"

'Monkeys wave goodbye' Há caracteres como, colocar aspas entre aspas, novas linhas, entre outros que não podem ser colocadas entre aspas, então a seguinte convenção é usada: Quando há uma barra invertida (\) dentro do texto entre aspas, indica que o caracter depois desta tem um significado especial. Uma aspa precedida de uma barra invertida não irá findar a string, e sim fazer parte dela. Quando o caracter 'n' correr depois da barra invertida, será interpretado como uma nova linha, e um 't', o caracter tab. Veja a string seguinte:

"This is the first line\nAnd this is the second"

O verdadeiro texto contido é:

This is the first line
And this is the second
